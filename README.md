# Memoire

A personal thought-capture app: jot down insights before they're lost, auto-filed into categories (health, psychology, self-help, finance, wisdom), and explored through an interactive word cloud.

Single self-contained HTML file — no build step, no dependencies.

## Deploy on GitHub Pages

1. Push this folder to a repository.
2. Repo **Settings → Pages → Source**: select your branch, root folder. Save.
3. Your app is live at `https://<username>.github.io/<repo>/`.

## Use on iPhone

Open the URL in Safari → **Share → Add to Home Screen**. Launches full-screen like a native app.

## Notes

- Thoughts are stored in the browser's local storage, per device (no sync).
- To update the app, replace `index.html` and push.
